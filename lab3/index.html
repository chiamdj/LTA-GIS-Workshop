    Lab Practice #3

    Learning Objectives

    In this lab exercise, you will learn the following:

    • Create and Export a map

    In this exercise, you will create two maps, one for Lab 1 and one for Lab 2.

    You should include the following map elements for both maps:

    • LTA logo (available in OneDrive folder)
    • Security classification (set as Restricted, Non-Sensitive)
    • Title
    • Your name and division
    • Legend
    • Scale bar
    • North arrow – Features should be labelled where applicable (e.g. bus stop number) – Features should be clearly visible on the map (tip: Use at least a size 9pt for point symbols)

    For Lab 2, use the Unique Value Symbology to use different colours to differentiate the different vehicles used (e.g. PAB, E-scooter etc.)

    Tip

    Feel free to add other map elements you feel is relevant or deemed fit.

    Help

    You may refer to Exercise 13 in the training handbook for the detailed steps.

    Using the layout in the previous step, create a map using the datasets used in Lab 1 and 2 and export it as a PDF with a resolution of 300 dpi.

    When you have completed this exercise, upload your PDF file into the folder Lab 3 submissions in the OneDrive folder where you downloaded your training materials.

    Label your PDF files as [Name]_Lab1.pdf and [Name]_Lab2.pdf.
